Bit scan reverse (stores index of first encountered 1 from the back).

BLENDVPS

Select packed single-precision floating-point values from xmm1 and xmm2/m128 from mask specified in XMM0 and store the values into xmm1 (SSE4.1).

CMOV_OVERFLOW

CMPPS

Compare packed single-precision floating-point value in xmm2/m32 and xmm1 using bits 2:0 of imm8 as comparison predicate (SSE).

CMP_IMM

Compare imm{16,32} with r/m{16,32,64} (sign-extended if 64).

CMP_IMM8

Compare imm8 with r/m{16,32,64}.

CMP_REG

Compare r{16,32,64} with r/m of the same size.

Divide packed single-precision floating-point values in xmm1 by packed single-precision floating-point values in xmm2/mem (SSE).

JUMP_SHORT_IF_OVERFLOW

Jump short (rel8) if overflow (OF=1).
