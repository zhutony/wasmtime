Variants

Unreachable

Explicit trap instruction

Block

Define metadata for a block - its label, its signature, whether it has backwards callers etc. It is an error to branch to a block that has yet to be defined.

Fields of Block

label: Labelparams: Vec<SignlessType>has_backwards_callers: boolnum_callers: Option<u32>
Label(Label)

Start a new block. It is an error if the previous block has not been closed by emitting a Br or BrTable.

Br

Unconditionally break to a new block. This the parameters off the stack and passes them into the new block. Any remaining elements on the stack are discarded.

Fields of Br

target: BrTarget<Label>

Returning from the function is just calling the "return" block

BrIf

Pop a value off the top of the stack, jump to the else_ label if this value is true and the then label otherwise. The then and else_ blocks must have the same parameters.

Fields of BrIf

then: BrTargetDrop<Label>

Label to jump to if the value at the top of the stack is true

else_: BrTargetDrop<Label>

Label to jump to if the value at the top of the stack is false

BrTable(BrTable<Label>)

Pop a value off the top of the stack, jump to table[value.min(table.len() - 1)]. All elements in the table must have the same parameters.

Call

Call a function

Fields of Call

function_index: u32
CallIndirect

Pop an i32 off the top of the stack, index into the table at table_index and call that function

Fields of CallIndirect

type_index: u32table_index: u32
Drop(RangeInclusive<u32>)

Pop an element off of the stack and discard it.

Select

Pop an i32 off of the stack and 2 elements off of the stack, call them A and B where A is the first element popped and B is the second. If the i32 is 0 then discard B and push A back onto the stack, otherwise discard A and push B back onto the stack.

Pick(u32)

Duplicate the element at depth depth to the top of the stack. This can be used to implement LocalGet.

Swap(u32)

Swap the top element of the stack with the element at depth depth. This can be used to implement LocalSet.
